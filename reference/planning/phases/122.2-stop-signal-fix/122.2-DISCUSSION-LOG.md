## Stop Reliability

| Option | Description | Selected |
|--------|-------------|----------|
| Hard stop (Recommended) | Wire dash:stop-task to call handleStopAutomation() which kills the iteration loop, cancels pending AI calls, cleans up content script observers. Instant halt. | Y |
| Graceful stop | Let the current action finish executing, then stop before the next iteration. Avoids mid-click interruption but adds 1-3s delay. | |
| Both with default | Single click = graceful, long-press/double-click = hard stop. More complex UI. | |

**User's choice:** Hard stop
**Notes:** User wants instant halt, no graceful wait.
