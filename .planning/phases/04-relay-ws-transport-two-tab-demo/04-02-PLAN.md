<objective>
Create the endpoint WebSocket transport that capture and viewer code can use without changing their fire-and-forget transport contract. Native `CompressionStream('deflate-raw')` is the default outbound compression path when available, legacy `{ _lz, d }` decode stays compatible with FSB, and async encoding is serialized behind a per-connection FIFO queue.

Purpose: RELY-02 closes the known Phase 4 ordering hazard and keeps the relay raw, stateless, and independently decodable.
Output: `src/transport/websocket.js` plus focused codec/ordering/status tests.

| Boundary | Description |
|----------|-------------|
| capture/viewer -> transport | Fire-and-forget sends enter an async codec queue |
| relay -> transport | Raw WebSocket data from the relay is decoded into structured messages |
| transport -> renderer/capture glue | Decoded type/payload pairs and status events are delivered to host handlers |
