<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| driven page -> adapter binding | Captured page code calls an exposed binding in the Node driver process. |
| viewer -> adapter | Viewer-originated control frames request privileged native input in a real browser. |
| adapter -> Playwright/CDP | Adapter crosses into driver-native input APIs that can affect the captured page. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | `validateRemoteControlMessage` | mitigate | Use 05-01 validators before every adapter control action. |
| T-05-02 | Information Disclosure | adapter events/state | mitigate | Emit summaries from 05-01 only; never log or emit raw bridge payloads or typed text. |
| T-05-03 | Elevation of Privilege | `requestControl` | mitigate | Default `authorizeControl` denies; call hook before active state and emit `denied` on false/error. |
| T-05-04 | Elevation of Privilege | `dispatchRemoteAction` | mitigate | Re-check `controlState === 'active'` immediately before every replay; inactive/denied dispatch count remains zero. |
| T-05-05 | Tampering | Playwright/CDP replay | mitigate | Use `page.mouse`, `page.keyboard`, or `CDPSession.send('Input.*')`; static tests forbid DOM synthetic replay. |
| T-05-06 | Information Disclosure | injected bridge | mitigate | Inject only in main frame, filter binding caller frame against `page.mainFrame()`, and restart capture after navigation. |
| T-05-07 | Tampering | coordinate replay | mitigate | 05-03 maps/rejects letterbox coordinates before viewer sends frames; adapter still validates finite non-negative coordinates. |
| T-05-08 | Spoofing | demo server | mitigate | 05-04/05-05 keep demo local-only and default-deny; adapter exposes no network listener itself. |
| T-05-09 | Tampering | relay boundary | mitigate | Adapter consumes frames from transport endpoints only; relay remains raw fan-out and never executes adapter logic. |
</threat_model>
