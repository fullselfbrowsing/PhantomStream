<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| viewer -> relay -> adapter | Untrusted viewer-originated control frames cross into the driver process. |
| protocol helpers -> telemetry | Potentially content-bearing control payloads are summarized for logs and state events. |
| relay core | Raw frame transport boundary must not become an execution or authorization boundary. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | `validateRemoteControlMessage` | mitigate | Reject unknown types, invalid coordinates, bad deltas, invalid buttons, invalid key events, and oversized text before adapter replay. |
| T-05-02 | Information Disclosure | `summarizeRemoteControlAction`, `createRemoteControlStateEvent` | mitigate | Redact typed text to character counts and omit `html`, `text`, `attrs`, `payload`, `url`, and `title` from summaries/state. |
| T-05-03 | Elevation of Privilege | adapter authorization | mitigate | This plan defines state constants consumed by 05-02; 05-02 enforces default-deny authorization before active state. |
| T-05-04 | Elevation of Privilege | adapter replay | mitigate | Validators produce normalized actions only; 05-02 checks active authorization again before every dispatch. |
| T-05-05 | Tampering | replay implementation | mitigate | Protocol tests establish action types only; 05-02 static tests forbid `dispatchEvent` and `element.click()` replay. |
| T-05-06 | Information Disclosure | injection bridge | mitigate | Main-frame filtering is implemented in 05-02; this plan keeps protocol payloads explicit and validated. |
| T-05-07 | Tampering | coordinate mapping | mitigate | 05-03 adds inverse mapping rejection for letterbox/out-of-bounds points; protocol rejects non-finite/negative coordinates. |
| T-05-08 | Spoofing | local demo | mitigate | 05-04/05-05 bind to `127.0.0.1`, print local URLs, and keep control default-deny. |
| T-05-09 | Tampering | `src/relay/relay.js` | mitigate | This plan adds a raw relay fan-out test for `REMOTE_CONTROL.CLICK`; relay does not inspect, authorize, or execute actions. |
</threat_model>
