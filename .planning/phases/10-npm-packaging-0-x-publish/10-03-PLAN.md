<task type="auto">
  <name>Task 1: Add package validators and tarball smoke script</name>
  <files>package.json, package-lock.json, scripts/package-smoke.mjs</files>
  <read_first>
    - `package.json`
    - `package-lock.json`
    - `.planning/phases/10-npm-packaging-0-x-publish/10-RESEARCH.md`
    - `tests/package-publish.test.js`
  </read_first>
  <action>
    Install dev dependencies `publint` and `@arethetypeswrong/cli`.

    Add `scripts/package-smoke.mjs` that:
    - runs `npm pack --json`;
    - parses the generated tarball path;
    - verifies packed file paths include `package/src/index.js`, `package/dist/types/index.d.ts`, `package/bin/phantom-stream.js`, `package/README.md`, and `package/LICENSE`;
    - verifies packed file paths do not include `package/.planning/`, `package/reference/`, `package/tests/`, or `package/.context/`;
    - creates a temp project with `npm init -y`;
    - installs the tarball;
    - imports every public subpath with Node ESM;
    - runs `node node_modules/.bin/phantom-stream --help`;
    - deletes the tarball and temp project on success or failure.

    Add scripts:
    - `"lint:package": "npm run types && publint"`
    - `"attw": "npm run types && attw --pack"`
    - `"package:pack": "npm run types && npm pack --dry-run"`
    - `"package:smoke": "npm run types && node scripts/package-smoke.mjs"`
    - `"package:check": "npm test && npm run types && npm run lint:package && npm run attw && npm run package:pack && npm run package:smoke"`
  </action>
  <verify>
    <automated>npm run package:check</automated>
  </verify>
  <acceptance_criteria>
    - `test -f scripts/package-smoke.mjs` passes.
    - `node -e "const p=require('./package.json'); for (const s of ['lint:package','attw','package:pack','package:smoke','package:check']) if(!p.scripts[s]) process.exit(1); for (const d of ['publint','@arethetypeswrong/cli']) if(!p.devDependencies[d]) process.exit(1)"`
    - `npm run package:check` exits 0.
  </acceptance_criteria>
  <done>Package validators and tarball install smoke pass locally.</done>
</task>
