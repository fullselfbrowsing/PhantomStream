<task type="auto">
  <name>Task 1: Add TypeScript declaration generation</name>
  <files>package.json, package-lock.json, tsconfig.types.json</files>
  <read_first>
    - `package.json`
    - `package-lock.json`
    - `.planning/phases/10-npm-packaging-0-x-publish/10-RESEARCH.md`
    - `tests/package-publish.test.js`
  </read_first>
  <action>
    Install `typescript` as a dev dependency.

    Add `tsconfig.types.json` with:
    - `"include": ["src/**/*.js", "bin/**/*.js"]`
    - `"compilerOptions.allowJs": true`
    - `"compilerOptions.checkJs": true`
    - `"compilerOptions.declaration": true`
    - `"compilerOptions.emitDeclarationOnly": true`
    - `"compilerOptions.declarationMap": true`
    - `"compilerOptions.outDir": "dist/types"`
    - `"compilerOptions.module": "NodeNext"`
    - `"compilerOptions.moduleResolution": "NodeNext"`
    - `"compilerOptions.target": "ES2020"`
    - `"compilerOptions.skipLibCheck": true`

    Add script `"types": "tsc -p tsconfig.types.json"`.
  </action>
  <verify>
    <automated>npm run types</automated>
  </verify>
  <acceptance_criteria>
    - `node -e "const p=require('./package.json'); if(!p.devDependencies.typescript) process.exit(1); if(p.scripts.types !== 'tsc -p tsconfig.types.json') process.exit(1)"`
    - `node -e "const fs=require('fs'); const c=JSON.parse(fs.readFileSync('tsconfig.types.json','utf8')); const o=c.compilerOptions; for (const k of ['allowJs','checkJs','declaration','emitDeclarationOnly','declarationMap']) if(o[k]!==true) process.exit(1); if(o.outDir!=='dist/types') process.exit(1)"`
    - `npm run types` exits 0.
  </acceptance_criteria>
  <done>Declaration generation is reproducible and source JS remains uncompiled.</done>
</task>
