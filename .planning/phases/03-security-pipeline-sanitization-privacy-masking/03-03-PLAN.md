## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| captured page -> wire | PII/secret content the host marked private must be transformed BEFORE transport.send — nothing downstream can un-leak it |
| host config -> capture core | Selectors and mask fns are host-supplied code/strings executing inside the serializer hot path |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-13 | Info Disclosure | password value via snapshot or attr-op (Pitfall 10) | mitigate | always-on el.type === 'password' mask in 'element', 'subtree', and attr-op 'value' shapes; non-configurable |
| T-03-14 | Info Disclosure | blocked-region content leaking via mutations on/inside the blocked element | mitigate | blockedWithAncestors ORed into all differ skip sites (including attr ops ON the blocked element); placeholder-only serialization; wire-wide absence scans in tests |
| T-03-15 | Info Disclosure | masked text leaking through the E2 text-childlist or add-op paths | mitigate | masking lives in the sanitizeForWire 'text' and 'subtree' shapes that 03-01 threaded through every path; live-ancestry check covers detached add-op clones |
| T-03-16 | Info Disclosure | throwing custom mask fn leaking raw text | mitigate | fail-closed containment: fallback to default asterisk mask, never raw (safeSkipElement precedent) |
