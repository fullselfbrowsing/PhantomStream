<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Host code -> capture handle | Trusted host asks for a live element's PhantomStream id. |
| Host code -> viewer handle | Trusted host asks renderer to resolve or highlight an id. |
| Viewer API -> host UI | Geometry and highlight state become public API output. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-07 | Information Disclosure | resolveNode API | mitigate | Tests reject `html`, `text`, `attrs`, `payload`, `url`, and `title` keys. |
| T-07-08 | Tampering | highlight API | mitigate | Highlight writes only host-local overlay DOM and never sends STREAM.OVERLAY frames. |
| T-07-09 | Denial of Service | stale ids | mitigate | Missing ids return null/false and do not throw during stream operation. |
</threat_model>
