| Boundary | Description |
|----------|-------------|
| page CSSStyleSheet methods -> capture wrappers | Page code invokes methods that capture wraps while streaming. |
| adoptedStyleSheets references -> scope registry | Page-controlled sheet references are reconciled into wire sources. |
| dynamic CSS text -> mutation payload | Runtime CSS updates are transported as style-source mutation ops. |

| T-09-18 | I | dynamic diagnostics | mitigate | Logs include only action/sourceId/reason; no CSS text, selector text, hrefs, or page content. |
| T-09-19 | D | dirty source batching | mitigate | rAF dedupe collapses repeated source changes and preserves relay cap bounding through existing `sendMutationDiffs`. |
