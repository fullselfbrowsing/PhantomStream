<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| CLI stdout -> user bookmark bar | Generated JavaScript is copied/executed by the user. |
| Bookmarklet loader -> source page | Loader runs in page context and can be blocked by CSP. |
| Source page -> local relay/viewer | Captured DOM leaves page through local WebSocket transport. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-13 | Tampering | Generated bookmarklet | mitigate | Reuse tested `createBookmarkletSource`; demo tests assert exact prefix and local URLs. |
| T-06-14 | Information Disclosure | Demo stdout | mitigate | Print only local URLs, bookmarklet source, and room prefix; no mirrored payloads. |
| T-06-15 | Denial of Service | Demo server | mitigate | Reuse strict local server close/path/no-store behavior. |
</threat_model>
