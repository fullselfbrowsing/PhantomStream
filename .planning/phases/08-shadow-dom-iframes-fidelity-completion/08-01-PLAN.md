| Boundary | Description |
|----------|-------------|
| captured page -> capture serializer | Untrusted page DOM, shadow content, and iframe elements enter PhantomStream payloads. |
| capture payload -> renderer sandbox | Serialized shadow/frame HTML is parsed and inserted into the mirror. |
| same-origin gate -> cross-origin frame | Browser origin policy decides whether capture may read iframe content. |
