| wire → relay | The new op rides the existing raw relay (1 MiB cap, envelope unchanged); old viewers must ignore it |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-14-02 | Tampering | A hostile/oversized hint payload exceeds the relay cap or corrupts the frame | mitigate | Task 1 raw round-trip test asserts encoded length `< RELAY_PER_MESSAGE_LIMIT_BYTES`; envelope is per-frame stateless (a bad frame cannot corrupt subsequent frames) |
