# Milestone v2.0 — Asset & Media Streaming — Audit

**Audited:** 2026-06-21
**Status:** tech_debt (no blockers; documented environment-forced real-browser UAT is the only outstanding item)

## Summary

Milestone v2.0 extended PhantomStream beyond DOM/text to mirror media **by reference**:
static assets, progressive `<video>`/`<audio>` with drift-corrected sync, best-effort
adaptive HLS/DASH with graceful fallback, and a completed media security contract — all
preserving the low-bandwidth core value (only URLs + small state messages cross the relay;
media bytes load from the source in the viewer's browser).

All 20 milestone requirements are **satisfied**, all 4 phases **verified**, cross-phase
integration is **sound** (every seam traced through code), and all 4 phases are **Nyquist
compliant**. The full test suite is **704/704** green and the differential oracle is **48/48**
(capture-wire byte-identity preserved). The published module stays **zero hard runtime
dependencies** (hls.js is an optional peerDependency, dynamic-import only).

The only outstanding item is **documented, environment-forced real-browser UAT** (15 items
across Phases 12–15): the test environment (jsdom + the FSB automation browser that runs tabs
hidden, suspending media) cannot exercise real subresource fetches, MSE/adaptive playback, CSP
enforcement, or referrer-header suppression. The backing logic of every one is unit-tested; the
never-break poster fallback is the safety net. These need a foregrounded/non-headless browser to
observe and are tracked in the per-phase HUMAN-UAT.md files.

## Cross-Phase Integration (sound)

The gsd-integration-checker traced every seam through the actual code (not summaries) and found
no wiring gap, payload-shape mismatch, contradictory security gate, or broken flow:

- **Capture → Protocol → Renderer media path:** `sendMediaState`/`buildMediaBaselineEntry` →
  `STREAM.MEDIA` → `handleMedia` → pure `reconcileMediaDrift` → `applyMediaAction`; `MediaSyncPayload`
  shape identical producer↔consumer.
- **Adapter discovery → renderer player:** `emitMediaHint` (`MediaHintPayload`) → `STREAM.MEDIA_HINT`
  → `handleMediaHint` (re-gates the manifest URL through the SAME Phase-12 `gateAssetUrl`) →
  `media-player.js` attach tree → single `degrade` sink.
- **Security gates end-to-end:** CSP layers cleanly (`default-src 'none'` + `img-src` [P12] +
  `media-src ... blob:` [P13/P14] + `referrer` no-referrer [P15], no `script-src`/`connect-src`);
  sandbox asserted exactly `allow-same-origin`; string-layer gate runs pre-srcdoc; masked media
  emits no `STREAM.MEDIA`.
- **Shared invariants:** envelope/relay byte-unchanged (verbatim raw forwarding); oracle 48/48;
  no new hard dependency.
- **Never-break chain:** every failure path (no-manifest/drm/mse-opaque/no-player/blocked-origin/
  masked) converges to poster/placeholder; re-snapshot teardown revokes object URLs (no leak).
